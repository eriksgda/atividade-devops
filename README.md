# Atividade DevOps e Tecnologias para desenvolvimento web

## Este projeto tem como objetivo apresentar todas etapas para conclusão das atividades somativas da matéria de DevOps e da matéria de Tecnologias para desenvolvimento web.

## Devido ao fato de ambas matérias se complementarem decidi realiza-las em um mesmo repositório, assim juntando desenvolvimento real com devOps.
